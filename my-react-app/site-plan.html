Overview

Purpose

My purpose for this site is to learn how to create an app using the React library. The other purpose for this site is to learn better programming skills that will help me later on as I search for jobs after I graduate from BYU-Idaho.

Audience

My audience for this project is young adults who are into video games. People love video games today, and they love to share their epic battles, massive losses, crippling defeats, and glorious victories.

Dynamic elements

The page will use React to display card information. The Sign in page will use a form that must be filled out in order to move into the "app". When the user clicks sign in, they are taken to the Media page.
